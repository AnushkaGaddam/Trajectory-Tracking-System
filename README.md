# Trajectory-Tracking-System
This project implements a trajectory tracking system that allows a robot  to follow a predefined path smoothly and accurately. The system simulates real-time movement while tracking a target trajectory and adjusting the robot’s position accordingly.

Features
Real-time trajectory/path tracking for robot or drone
Smooth movement simulation along a predefined path
3D visualization of the robot movement
Sonar radar scanning system for environment awareness
Collision warning system for nearby obstacles
Underwater fog/turbidity simulation for realistic environment conditions
Battery level simulation for the drone
Obstacle detection and avoidance visualization
Graphical path display showing the robot’s trajectory

Tech Stack
Python – Core programming language
VPython – 3D visualization and simulation
NumPy – Mathematical computations
Matplotlib – Graph plotting and visualization
